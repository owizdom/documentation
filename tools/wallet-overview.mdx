# Wallet Overview Tool

The `getWalletOverviewTool` gives you a complete picture of any wallet on Ethereum or Base, including smart wallet detection, token balances, and activity metrics.

## What It Does

- **Wallet Type Detection** - Identifies EOA, multisig, smart wallets, and delegated accounts
- **Token Balances** - All token holdings with USD values
- **Activity Metrics** - Transaction counts and deployed contracts
- **Multisig Details** - Signers and pending transactions for Safes
- **ERC-7702 Support** - Detects delegated EOAs and their implementations

## How to Use It

Ask Claude about any wallet:

```
Give me an overview of vitalik.eth
```

```
What tokens does 0x... hold?
```

```
Is this wallet a multisig? Who are the signers?
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `walletAddress` | Yes | The wallet address (0x...) |
| `blockchain` | No | `ethereum` (default) or `base` |

## Wallet Types Detected

| Type | Description |
|------|-------------|
| **EOA** | Externally Owned Account (regular wallet) |
| **ERC-7702** | EOA with delegated code (smart account features) |
| **ERC-4337** | Smart contract wallet (Account Abstraction) |
| **Multisig** | Safe or similar multi-signature wallet |
| **Contract** | Other smart contract |

## Practical Use Cases

### Due Diligence on a Wallet

**You:** "Is this wallet controlled by a single person or is it a multisig?"

**Claude identifies** the wallet type and provides relevant details about control.

### Portfolio Analysis

**You:** "What's the total value of tokens in this wallet?"

**Claude returns** all balances with USD values and totals.

### Investigating a DAO Treasury

**You:** "Give me details on the Uniswap governance treasury"

**Claude shows:**
- Multisig configuration
- Signers
- Major holdings
- Pending transactions

### Checking Wallet Safety

**You:** "Is it safe to receive funds from this address?"

**Claude analyzes:**
- Wallet type
- Historical activity
- Any red flags

## Common Questions

<AccordionGroup>
  <Accordion title="What if the wallet has many tokens?">
    The overview shows the most valuable holdings. Ask for "all tokens" to see the complete list.
  </Accordion>

  <Accordion title="How current are the balances?">
    Balances reflect the latest indexed block, typically within seconds of real-time.
  </Accordion>

  <Accordion title="Can I see NFTs?">
    Yes, NFT holdings are included with floor price estimates where available.
  </Accordion>
</AccordionGroup>

## Limitations

<Warning>
**USD values are estimates.** Based on current market prices and may fluctuate.
</Warning>

<Warning>
**Some tokens may not have prices.** Low-liquidity or new tokens might show $0 value.
</Warning>
