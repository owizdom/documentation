# Token Activity Tool

The `getTokenActivityTool` provides detailed token balance and transfer history for any address. Perfect for tracking specific token movements and analyzing holding patterns.

## What It Does

- Shows current balance of a specific token
- Lists transfer history (sent and received)
- Indicates transfer direction (in/out)
- Supports pagination for full history

## How to Use It

Ask Claude about token activity:

```
Show me the USDC transfer history for 0x...
```

```
When did this wallet last receive ETH?
```

```
Track PEPE token activity for this address
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `holderAddress` | Yes | The wallet address to check |
| `tokenAddress` | Yes | The token contract address |
| `blockchain` | No | `ethereum` (default) or `base` |
| `page` | No | Page number (default: 1) |

<Note>
You can use token symbols (like "USDC") - Claude will look up the correct contract address.
</Note>

## Understanding Direction

| Direction | Meaning |
|-----------|---------|
| **IN** | Tokens received by the holder |
| **OUT** | Tokens sent from the holder |

## Practical Examples

### Example 1: Tracking Incoming Funds

**You:** "When did this wallet last receive USDC and from where?"

**Claude finds:**
```
Last USDC received:
- Date: 2024-01-14 10:15:00
- Amount: 5,000.00 USDC
- From: 0xCoinbase... (Coinbase Exchange)
```

### Example 2: Analyzing Spending Patterns

**You:** "How much ETH has this wallet spent in the last month?"

**Claude calculates:**
```
ETH Outflows (Last 30 Days):
- Total sent: 15.5 ETH ($49,000)
- Number of transfers: 23
- Largest single transfer: 5 ETH to 0xDeFi...
- Most frequent recipient: 0xUniswap...
```

### Example 3: Tracking Whale Movements

**You:** "Track PEPE transfers for this known whale wallet"

**Claude shows:**
```
PEPE Activity Summary:
- Current balance: 50B PEPE ($450,000)
- Accumulated: 75B PEPE total bought
- Distributed: 25B PEPE sold/transferred
- Average hold time: 45 days
```

## Pagination

For addresses with extensive history:

**You:** "Show page 5 of USDC transfers for 0x..."

```
Transfer History (Page 5):
Showing transfers 401-500 of 1,234
```

## Comparing Multiple Tokens

You can ask about multiple tokens:

**You:** "Compare the USDC and USDT activity for this wallet"

**Claude will make multiple queries and provide comparison.**

## Combining with Other Tools

For complete analysis:

1. **Check all holdings** → `getWalletOverviewTool`
2. **Focus on one token** → `getTokenActivityTool`
3. **Analyze specific transfer** → `queryTransactionTool`
4. **Check counterparty** → `getWalletOverviewTool` on the other address

## Tips

<Tip>
**Use token symbols.** "USDC activity" is easier than typing the full contract address.
</Tip>

<Tip>
**Ask for summaries.** "Summarize the token flow" gives you insights, not just raw data.
</Tip>

<Tip>
**Check both directions.** "What's the net flow of USDC for this wallet?" considers both in and out.
</Tip>

## Supported Token Types

| Type | Support |
|------|---------|
| ERC-20 | Full support |
| Native ETH | Full support |
| ERC-721 (NFTs) | Transfer tracking |
| ERC-1155 | Transfer tracking |

## Limitations

<Warning>
**100 transfers per page.** Use pagination for complete history.
</Warning>

<Warning>
**Historical depth.** Very old transfers may have limited data.
</Warning>
