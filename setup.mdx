# Setup

## Install Claude Code

<Tabs>
  <Tab title="macOS">
```bash
brew install claude-code
```
  </Tab>
  <Tab title="Linux">
```bash
curl -fsSL https://claude.ai/install.sh | sh
```
  </Tab>
  <Tab title="Windows">
Download from [claude.ai/claude-code](https://claude.ai/claude-code)
  </Tab>
</Tabs>

Verify:

```bash
claude --version
```

## Get API Key

1. Sign up at [herd.eco](https://herd.eco)
2. Navigate to dashboard
3. Copy API key

<Warning>
Keep your API key private. Never commit to version control.
</Warning>

## Add Herd MCP

```bash
claude mcp add --transport http herd https://api.herd.eco/v1/mcp \
  --header "Authorization: Bearer YOUR_API_KEY"
```

### Using Environment Variables

```bash
# Add to ~/.zshrc or ~/.bashrc
export HERD_API_KEY="your_key"
```

```bash
claude mcp add --transport http herd https://api.herd.eco/v1/mcp \
  --header "Authorization: Bearer $HERD_API_KEY"
```

### Multiple Configurations

```bash
# Personal
claude mcp add --transport http herd-personal https://api.herd.eco/v1/mcp \
  --header "Authorization: Bearer $HERD_PERSONAL"

# Work
claude mcp add --transport http herd-work https://api.herd.eco/v1/mcp \
  --header "Authorization: Bearer $HERD_WORK"
```

## Manage

```bash
# List configured MCPs
claude mcp list

# Remove
claude mcp remove herd
```

## Available Tools

| Tool | Purpose |
|------|---------|
| `contractMetadataTool` | Contract info, ABI, proxy history, token data |
| `regexCodeAnalysisTool` | Search contract source code |
| `diffContractVersions` | Compare implementation versions |
| `queryTransactionTool` | Decode transactions with traces and logs |
| `getLatestTransactionsTool` | Recent function calls or events |
| `getWalletOverviewTool` | Wallet type, balances, tx count |
| `getTokenActivityTool` | Token balance and transfer history |
| `getTransactionActivityTool` | Address transaction history |
| `getDeployedContractsTool` | Contracts deployed by address |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| MCP not found | Check spelling with `claude mcp list` |
| Auth error | Verify API key, no extra spaces |
| Timeout | Check internet connection |
| Tool not responding | Remove and re-add MCP |
