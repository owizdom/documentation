# Quick Reference

## Tool Summary

| Tool | Use For | Example Query |
|------|---------|---------------|
| `contractMetadataTool` | Contract info, ABI, tokens | "What is contract 0x...?" |
| `regexCodeAnalysisTool` | Search contract code | "Find the transfer function" |
| `diffContractVersions` | Compare upgrades | "What changed in the upgrade?" |
| `queryTransactionTool` | Decode transactions | "What happened in tx 0x...?" |
| `getLatestTransactionsTool` | Recent function/event calls | "Show recent swaps" |
| `getTransactionActivityTool` | Address tx history | "What has 0x... been doing?" |
| `getWalletOverviewTool` | Wallet profile | "Overview of 0x..." |
| `getTokenActivityTool` | Token transfer history | "USDC history for 0x..." |
| `getDeployedContractsTool` | Find deployments | "What contracts did 0x... deploy?" |

## Common Queries

### Contracts

```
# Get contract overview
What is the contract at 0x...?

# Check if upgradeable
Is 0x... an upgradeable proxy?

# Get ABI
What functions does 0x... have?

# Search code
Find the swap function in 0x...

# Compare versions
What changed in the latest upgrade of 0x...?
```

### Transactions

```
# Basic decode
What happened in transaction 0x...?

# Detailed analysis
Show me all balance changes in tx 0x...

# Find specific transfers
Show only USDC transfers in tx 0x...

# Find function calls
Show me recent deposit() calls on 0x...

# Find events
Show recent Transfer events from 0x...
```

### Wallets

```
# Full profile
Give me an overview of 0x...

# Check holdings
What tokens does 0x... hold?

# Transaction history
Show recent transactions from 0x...

# Token history
Show USDC transfer history for 0x...

# Deployments
What contracts has 0x... deployed?
```

## Query Modifiers

### Specifying Chain

```
# Default is Ethereum
What is 0x...?

# For Base
What is 0x... on Base?
```

### Pagination

```
# First page (default)
Show transactions for 0x...

# Specific page
Show page 3 of transactions for 0x...
```

### Filtering

```
# Filter balance changes
Show only ETH transfers in tx 0x...

# Filter by destination
Show transactions from 0x... to Uniswap
```

## Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| "Contract not found" | Check address and chain |
| "No source code" | Contract not verified |
| "Transaction not found" | Check hash and wait for confirmation |
| "Rate limit" | Wait and try again |

## Keyboard Shortcuts (Claude Code)

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Cancel current operation |
| `Ctrl+L` | Clear screen |
| `↑` / `↓` | Navigate history |

## Best Practices

<Tip>
Be specific: "What does the swap function do?" beats "explain this contract"
</Tip>

<Tip>
Mention the chain—always specify if you're on Base.
</Tip>

<Tip>
Start broad, then narrow. Get overview first, then drill into details.
</Tip>
