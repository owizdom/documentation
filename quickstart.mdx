# Quickstart

## Prerequisites

- Claude Code installed
- Herd API key from [herd.eco](https://herd.eco)

## 1. Add Herd MCP

```bash
claude mcp add --transport http herd https://api.herd.eco/v1/mcp \
  --header "Authorization: Bearer YOUR_API_KEY"
```

Replace `YOUR_API_KEY` with your actual key.

## 2. Verify

Start Claude Code and run:

```
What Herd tools are available?
```

Expected response includes `contractMetadataTool`, `queryTransactionTool`, and others.

## 3. Test

```
Get metadata for USDC: 0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48
```

Claude returns contract info, ABI summary, and token data.

## Quick Reference

| Task | Example Query |
|------|---------------|
| Contract info | `What is contract 0x...?` |
| Decode transaction | `What happened in tx 0x...?` |
| Wallet overview | `Show holdings for 0x...` |
| Search code | `Find mint functions in 0x...` |
| Compare versions | `Diff the last two implementations of 0x...` |
